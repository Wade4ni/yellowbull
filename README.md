# yellowbull(以Momo購物網為例)

## 使用方法:
1.打開**Anaconda Prompt**安裝必要套件:
<br>
```pip install webdriver_manager```
<br>
```pip install selenium```

2.將**buy_bot.ipynb**下載並放至使用者資料夾
<br>
<br>
3.執行檔案

```僅初次使用需要執行上半程式碼，需要紀錄使用者登入資訊，以便後續做自動登入的動作，詳細說明請見程式碼註解，後續進行程式下單只需要進行下半部程式碼```
